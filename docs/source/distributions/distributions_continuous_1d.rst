Continuous Distributions 1D
---------------------------------------

In :py:mod:`UQpy`, univariate continuous distributions inherit from the :class:`.DistributionContinuous1D` class:

Since the continuous distributions are based on :py:mod:`scipy`, all of the distribution types described posses the
following methods: :py:meth:`cdf`, :py:meth:`pdf`, :py:meth:`log_pdf`, :py:meth:`icdf`, :py:meth:`rvs`,
:py:meth:`moments`, :py:meth:`fit`
